# Kekomê? 🍰 Tem que fazer! 🥣

App de cadastro de receitas onde o usuário poderá adicionar:
- Descrição;
- Ingredientes;
- Modo de preparo;
- Rendimento.

E visualizar, editar e excluir as receitas adicionadas anteriormente.
